# 에러 코드 체계

## 에러 코드 목록

| 코드 | HTTP 상태 | 메시지 | 발생 조건 |
|------|----------|--------|----------|
| AUTH_001 | 401 | 인증이 필요합니다 | 토큰 누락 |
| AUTH_002 | 403 | 권한이 없습니다 | 타인 데이터 접근 |
| VAL_001 | 400 | 금액이 올바르지 않습니다 | 음수 또는 0 금액 |
| VAL_002 | 400 | 카테고리가 올바르지 않습니다 | 허용되지 않은 카테고리 |
| EXPENSE_001 | 404 | 지출을 찾을 수 없습니다 | 존재하지 않는 expenseId |
| BUDGET_001 | 409 | 해당 월 예산이 이미 존재합니다 | 중복 월 예산 생성 |
| SERVER_001 | 500 | 서버 오류가 발생했습니다 | 내부 예외 |
