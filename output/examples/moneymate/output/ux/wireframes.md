# Wireframes

## 대시보드

- 상단: 월 선택, 남은 예산 카드, 총지출 카드
- 중단: 카테고리별 소비 막대 그래프
- 하단: 최근 지출 리스트와 주간 인사이트
- 모바일: 카드 스택 구조
- 데스크톱: 2열 레이아웃
- empty: 아직 지출이 없으면 첫 입력 CTA 표시
- loading: 요약 카드 skeleton
- error: 다시 불러오기 버튼

## 지출 내역

- 리스트 + 우측 또는 하단 입력 폼
- 빠른 카테고리 선택, 금액 입력, 날짜 선택
- 모바일: 하단 시트 입력
- 데스크톱: 고정 입력 패널

## 예산 설정

- 월간 한도 입력
- 현재 예상 잔여 예산 미리보기
- 저장 후 대시보드로 이동
