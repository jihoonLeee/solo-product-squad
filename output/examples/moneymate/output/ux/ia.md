# IA

## 사이트맵

- 공개
  - 랜딩
  - 로그인
  - 회원가입
- 인증 후
  - 대시보드
  - 지출 내역
  - 예산 설정
  - 인사이트
  - 설정

## 주요 사용자 흐름

1. 회원가입
2. 월간 예산 입력
3. 첫 지출 3건 기록
4. 대시보드에서 잔여 예산 확인
5. 인사이트에서 초과 지출 카테고리 확인

## 페이지별 역할

- 랜딩: 문제와 가치 제안 전달
- 대시보드: 잔여 예산과 주요 수치를 즉시 표시
- 지출 내역: 입력, 수정, 삭제 중심
- 예산 설정: 월간 목표 관리
- 인사이트: 행동 추천 제공
